# Описание сервисов

## 1. Telegram Bot  
- Интерфейс пользователя через Telegram API.  
- Команды: регистрация, просмотр анкет, лайк/пропуск, мэтчи, приглашения.  
- Общается с другими сервисами через HTTP и Message Broker.

## 2. Profile Service (анкеты)  
- CRUD анкет: возраст, пол, интересы, предпочтения, фото (загрузка в S3).  
- Хранит данные в PostgreSQL.

## 3. Ranking Service (ранжирование)  
- Считает первичный рейтинг (анкета + полнота), поведенческий (лайки, мэтчи, диалоги) и комбинированный.  
- Сохраняет рейтинги в БД.  
- По запросу выдаёт отсортированный список анкет для пользователя.

## 4. Redis (кэш)  
- Хранит готовые списки анкет (например, следующие 10) для быстрой выдачи.  
- Обновляется при пересчёте рейтингов.

## 5. Message Broker (Kafka/RabbitMQ)  
- Асинхронная передача событий: лайки, пропуски, мэтчи, изменение анкет.  
- Разгружает синхронные вызовы.

## 6. Event Processor  
- Слушает события, обновляет метрики пользователей (количество лайков, мэтчей и т.д.) в БД.  
- Запускает пересчёт рейтинга при необходимости.

## 7. Celery (фоновые задачи)  
- Периодический пересчёт рейтингов.  
- Обновление кэша в Redis.  
- Отложенная обработка фото.

## 8. Referral Service  
- Реферальная система: учёт приглашённых друзей, начисление бонусов к рейтингу.

## 9. База данных (PostgreSQL)  
- Таблицы: пользователи, анкеты, фото, рейтинги, метрики, взаимодействия, рефералы.

## 10. S3-хранилище (Minio)  
- Хранение фотографий пользователей.

## 11. Мониторинг и логирование  
- Prometheus + Grafana для метрик.
